How does a website work?

hello, guys today's topic is about how does a website work. In today's world, everyone is using the internet for watching youtube videos, searching on google for unknown information or things.but when we are using google for knowing the unknown information we get a bunch of websites showing related information. but have you wondered how does this all process from searching for information to getting the result of website works. So the entire process will be explained in an easy way, just stay tuned.

Process 1:

We use a web browser for surfing the internet and there are a lot more web browsers available on the internet like Chrome, Mozilla, etc. So let us chose the chrome browser for searching the information or unknown thing. when we type information in the search bar of the browser it will display a lot of websites.



Process 2:

let take an example of typing facebook.com in the search bar, now the browser sent data or request to the DNS server. wait what is DNS server you might be wondering, DNS server is nothing like phone book directory which contains domain name(facebook.com) mapped to its IP address(213.23.44.55). An IP address is a unique address given to a server or device to identify itself on the internet just like your home address.



Process 3:

Now DNS server search for the IP address of the requested domain name (i.e facebook.com) and it sent the IP address to the browser if present or else it sent a request to other DNS server for the IP address. Now the browser directly reaches facebook.com using an IP address of which it has received.



Process 4:

Once the facebook server receives a request from the browser. now the facebook server sends the login page to the browser in the form of an HTML file, which browser understands and displays in the browser.
